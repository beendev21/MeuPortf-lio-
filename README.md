## 🚀 Sobre o Projeto

Este projeto consiste no desenvolvimento do meu portfólio pessoal, criado com o objetivo de reunir e exibir minhas habilidades, projetos e trajetória profissional na área de tecnologia.

A aplicação foi construída com foco em uma interface moderna, responsiva e intuitiva, permitindo uma navegação clara e objetiva. Além de funcionar como vitrine profissional, o projeto também demonstra na prática minha evolução como desenvolvedor e aplicação de boas práticas de desenvolvimento.
